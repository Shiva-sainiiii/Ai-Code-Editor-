# Nexus AI — AI Powered Code Editor 🚀

Nexus AI is a modern browser-based AI code editor built for developers who want speed, simplicity, and smart assistance. Inspired by tools like Cursor AI and VS Code, it combines coding, AI help, live preview, and cloud saving into one clean workspace.

This project reflects practical frontend development, backend API integration, and real-world developer tools architecture.

---

## ✨ Features

### 💻 Smart Code Editor
- Monaco Editor integration
- Syntax highlighting
- Multi-language support
- Fast coding environment
- Clean responsive UI

### 🤖 AI Assistant
- Generate code
- Debug errors
- Explain code
- Optimize logic
- Productivity support

### ⚡ Live Preview
- Run HTML / CSS / JavaScript instantly
- Real-time output window

### ☁️ Cloud Save
- Save projects online
- Open anytime
- Firebase Firestore integration

### 🧩 Productivity Tools
- Copy
- Save
- Open
- Undo / Redo

### 📱 Responsive Design
- Mobile friendly
- Desktop optimized
